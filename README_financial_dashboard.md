## 📌 Overview

A full-featured analytics dashboard that analyzes financial transaction data — providing real-time insights into transaction volumes, fraud patterns, customer behavior, and operational KPIs. Built to simulate the kind of dashboards used by fintech and banking data teams.

\---

\---

## ✨ Features

* **6 KPI Metrics** — Total transactions, volume, average amount, fraud rate, success rate
* **Monthly Trend Charts** — Volume and count over time with dual Y-axis
* **Fraud Intelligence** — Fraud cases per month, fraud rate by category, amount distribution
* **Customer Insights** — Spending by gender, age distribution, top cities by volume
* **Behavioral Heatmap** — Transaction patterns by day of week and hour
* **Fraud Alerts Table** — Most recent fraudulent transactions
* **Auto-Generated Insights** — Peak hours, busiest days, highest risk categories
* **Sidebar Filters** — Filter by month, category, and amount range

## 🚀 Getting Started

### 1\. Clone the Repository

```bash
git clone https://github.com/YOUR\_USERNAME/financial-analytics-dashboard.git
cd financial-analytics-dashboard
```

### 2\. Install Dependencies

```bash
pip install streamlit pandas numpy plotly
```

### 3\. Run the App

```bash
streamlit run jazzcash\_dashboard.py
```

Open `http://localhost:8501` — upload your CSV or explore with built-in sample data!
